# 駅エリア不動産価値分析ツール - 要件定義

## 📋 プロジェクト概要

首都圏の駅単位で不動産価値の過去推移と将来予測を表示するウェブアプリケーション。

ユーザーは駅名を検索し、その駅周辺の不動産価値の時系列推移、周辺開発情報、AI による将来価格予測を確認できます。

## 🎯 目的

- 起業・脱サラのための不動産投資判断を支援
- 将来性の高い駅・エリアの発掘
- 周辺開発情報を加味した価値判断

---

## 🛠️ 技術構成

| 項目 | 技術 |
|------|------|
| **フロントエンド** | Next.js / TypeScript |
| **スタイリング** | Tailwind CSS |
| **グラフ表示** | Recharts |
| **デプロイ環境** | Vercel |
| **データ更新** | 週次自動更新（キャッシュ機構） |

---

## 🗺️ 対象エリア

- **東京都**
- **神奈川県**
- **千葉県**
- **埼玉県**

※ 都内アクセスが良好なエリアを優先

---

## 📊 データソース

| データ項目 | ソース | 更新頻度 |
|---------|--------|--------|
| 不動産取引価格 | 国土交通省 不動産取引価格API | 週次 |
| 駅・路線情報 | 国土数値情報 | 週次 |
| 周辺開発情報 | ニュースAPI | 週次 |

## 🎨 UI/UX 要件

### 駅検索画面
- テキスト入力による駅名検索
- オートコンプリート表示
- 複数駅の比較機能（将来実装）

### 詳細分析画面
- **過去データセクション**
  - 5年/10年/全期間の推移グラフ
  - 価格上昇率の表示

- **周辺開発情報セクション**
  - 最新ニュース一覧
  - 開発プロジェクト情報

- **AI 予測セクション**
  - 向こう 3～5年の価格予測グラフ
  - トレンドスコア（0～100）
  - リスク評価

### デザイン方針
- ダークトーン（夜間利用を想定）
- シック・プロフェッショナルな見た目
